Analizamos nuestro conjunto de datos risk-factors y la variable edad es una variable que posee todos lo datos, pero tenemos otras variables en donde existen datos faltantes.

Lo que yo quiero hacer es:

Si yo tengo todos los valores de la edad, habr valores faltantes asociados a ciertas edades? pues podemos averiguarlo haciendo grficas como las siguientes.

En la anterior grfica agregamos una nueva variable weight_lbs_NA tiene 2 clasificaciones Not Missing y Missing (Es decir si el peso es nulo o no - Si est faltante o no). De esta manera podemos determinar si los valores faltantes estn ocurriendo en cierta zona referente a otra variable o no y de esta manera obtener estadsticos que nos ayuden a entender que est pasando con estos valores faltantes y asociarlo posiblemente a algn tipo de mecanismo de valores faltantes.

En la grfica anterior podemos observar, entonces podemos decir que de los valores faltantes nuestros datos faltantes se encuentran entre la zona de 60 y su media es superior a 60. Tambin vemos que existen un sesgo en la distribucin y que los cuantiles estn desproporcionados. Es decir tenemos una zona donde se concentra la distribucin y existencia de outliers.

Entonces tambin de esta forma podemos hacer una relacin de que el peso de los pacientes este relacionado con la edad, pero necesitamos explorar ms a fondo para asegurarlo.

Pero lo ms importante es cmo llegamos nosotros a esta grfica y establecer este tipo de relaciones? usaremos el concepto de shadow matrix

Shadow matrix

Cmo la vamos a construir?

  1. Identificar todos los valores faltantes, es decir vemos que hay N/A.
  2. Lo siguiente es reemplazarlos.
  3. Al ltimo reemplazar por algo ms informativo como NA, para que la librera pueda leer los valores faltantes. Ademas de aadir un sub-fijo NA para cada variable y as identificar los valores faltantes. As voy a tener mi variable con un sub-fijo elegido

Caractersticas de la matriz de sombras

  • Nombres coordinados.
  • Valores explcitos.

De esta manera tengo mis datos, es decir:

  • Tabla original.
  • Matriz de sombras
  • La fusin de ambas en una gran matriz que se le conoce como Nabular. Que viene de la combinacin de NA (Datos faltantes) y Bular (Tabular ) de las tablas originales.

Entonces tendramos nuestra matriz y los valores faltantes lista para realizar visualizaciones importantes. Vamos a crear una con cdigo.

(
    riskfactors_df
    .isna()

        False:'Not missing',
        True: 'Missing'

    .add_suffix('_NA')
    .pipe(

            [riskfactors_df,shadow_matrix],
            axis = 'columns' 
            #Le decimos que queremos unir por columnas
        )
    )
