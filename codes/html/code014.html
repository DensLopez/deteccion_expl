Una vez que ya tenemos nuestra matriz de sombras. Procederemos a realizar lo siguiente:

In[14]:
(
    riskfactors_df
    .missing.bind_shadow_matrix(only_missing=True)
    .pipe(
        lambda df: (
            sns.boxenplot(
                data=df,
                x='weight_lbs_NA',
                y='age', #Variable que queramos observar su distribucin
                hue = 'weight_lbs_NA'
            )
        )
    )
)
Out[14]:
<AxesSubplot:xlabel='weight_lbs_NA', ylabel='age'>

Hemos recreado la imagen que estaba en las diapositivas, adems digamos que es muy potente el poseer la matriz de sombras para obtener estadsticos y aplicar los anlisis que hemos hecho para los datos, pero esta vez con datos faltantes.

In[18]:
(
    riskfactors_df
    .missing
    .bind_shadow_matrix(only_missing=True)
    .pipe(

            sns.displot(
                data=df,
                x='age',

            )
        )
    )
)

Podemos ver la distribucin de nuestros pesos weight_lbs_NA la que tenemos en la densidad y podemos ver donde se estn acumulando la de nuestros valores faltantes. Vemos que se repite la distribucin simplemente que de una manera ms pequea. Ademas en la azul tenemos un pico por debajo de los 60 y en la naranja (faltantes) un pico por encima de los 60.

Aadiremos un parmetro extra.

In[22]:
(
    riskfactors_df
    .missing
    .bind_shadow_matrix(only_missing=True)
    .pipe(
        lambda df: (
            sns.displot(
                data=df,
                x='age',
                col = 'weight_lbs_NA', 
                hue='weight_lbs_NA',
                facet_kws={'sharey':False}
            )
        )
    )
)
Out[22]:
<seaborn.axisgrid.FacetGrid at 0x7f2db1856d90>

El parmetro sharey = False va a hacer que cada grfica tenga un valor diferente en el eje Y, as de esta manera se ajusta para que los valores se normalicen y visualmente sean del mismo tamao. Es decir que sus ejes y tengan diferente escala.

Aqu podramos aplicar ms conceptos de exploracin de datos; modificar tamao de barras o asociar otras variables para saber si estn ocurriendo algunos comportamientos en los datos.
