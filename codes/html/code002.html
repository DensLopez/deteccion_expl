Operaciones con valores faltantes

Ahora que sabemos sobre la problemtica con los valores faltantes, vamos a aprender sobre las operaciones y por qu es que son importantes?

Tenemos una operacin sencilla

Pero qu pasa si en algn momento nos falta un valor?, entonces qu debera ser el resultado de est operacin?

  1. El resultado ser otro valor faltante es decir algo desconocido.
  2. Este otro resultado podra ser un Warning de que esto no est funcionando como debera, la operacin no est permitida de cierta manera o algo as.
  3. Finalmente lo ltimo sera que el software no puede trabajar con esto. Y me dara un error sobre el resultado.

Estas son distintas opciones y todas ellas son validas dependiendo de que el software elija cual regresar.

Por ejemplo podemos observar 2 cdigos: Python y R.

En Python realizamos todo el proceso y vemos que tenemos un valor faltante None. Si le aplicamos la media, obtenemos como resultado un 3, con esto podemos deducir que lo est ignorando.

En R realizamos un proceso similar pero el resultado de este ser un valor faltante (NA).

Entonces a pesar de que las herramientas estn ejecutando la misma funcin, los resultados son diferentes.

Recordemos que:

As podemos hacer una comparacin diciendo que cada software es distinto y adentrarnos en el para aprender como tratar los valores faltantes.

Numpy

Es una de las formas con las que vamos a trabajar los valores faltantes.
La forma en la que Numpy los representa es a traves de np.nan

    sep='\n'

nan
In[]:

nan

True

Esto es debido a cmo esta funcionando Python y cmo Numpy est manejando estos valores faltantes. Pero si ahora intentamos realizar operaciones con los valores nulos.

In[]:
print(
    np.nan/2,
    type(np.nan),
    sep='\n'
)

nan
<class 'float'>

Podemos ver que con Numpy si se puede realizar operaciones con valores faltantes.

Por qu sucede?
Sucede porque si nosotros verificamos el tipo de dato que es nan , el resultado es que es de la clase float. Esto quiere decir que es un nmero y que puedo realizar operaciones con el a pesar de que me regrese como resultado un nan

Aqu podemos ver como esta representando los valores faltantes. Qu esta pasando y por qu Pandas tiene tantas formas de representar los valores faltantes?

Sucede que Pandas est en una constante evolucin y hace que se representen como datos faltantes y no como float.

Tomando en cuenta que Pandas a estos datos
